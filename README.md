# recruiting
### 解题思路
## 题意
>  给出停车的时间起点和终点（终点不计入停车时间）

   求上午最多停车辆数，要求时间复杂度不高于：O(n)*lgN。
 
## 解题思路
>  由于时间都为整点，上午的整点数12个，所以采用桶计数，换取时间。

   创建一个数组，表示某小时（下标）内最大停车数，初始化为0。
   
   首先排除无效记录，再遍历有效记录并在数组中增加车辆数。
   
   最后找出上午数组中的最大值即可。
